# 💪 Sahil Workout System

Professional workout dashboard with dedicated programs for:
- Chest
- Back
- Shoulders
- Legs & HIIT

## Live Deployment

### GitHub Pages
1. Create a GitHub repository.
2. Upload all files.
3. Go to Settings → Pages.
4. Deploy from `main` branch.

### Netlify
Drag and drop the project folder or connect GitHub.

### Vercel
Import the GitHub repository and deploy.
